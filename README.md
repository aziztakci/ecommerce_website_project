## ECOMMERCE WEBSITE PROJECT
A professional, high-performance e-commerce application built with **React 19** and **Tailwind CSS v4.2**.

## 🚀 Live Demo
You can view the live project here: https://ecommerce-website-project-wine.vercel.app/

## 🛠 Tech Stack
* **Framework:** React 19 (Vite)
* **Styling:** Tailwind CSS v4.2 (Utility-first)
* **State Management:** Legacy Redux & Redux Thunk
* **Routing:** React Router v7
* **Utilities:** Axios, React Toastify

## 🏗 Key Features (Planned)
* Responsive Design for all devices
* State-managed Product Catalog
* Advanced Shopping Cart functionality
* Seamless Navigation with React Router
